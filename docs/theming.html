Theming

Customize the look of your terminal UI with built-in themes or create your own.

Using Themes

Set the theme using the data-theme attribute on any parent element:

<!-- On the html element (recommended) --> <html data-theme="dark"> <!-- Or on body --> <body data-theme="matrix-green"> <!-- Or on a specific container --> <div data-theme="retro-amber"> <!-- This section uses retro-amber theme --> </div>

Creating Custom Themes

Create your own theme by defining the CSS custom properties:

/* Define a new theme */ [data-theme="cyberpunk"] { --tui-bg: #0a0a0f; --tui-fg: #f0f0f0; --tui-accent: #ff00ff; --tui-panel: #1a1a2e; --tui-grid-line: rgba(255, 0, 255, 0.2); --tui-dim: #888; --tui-success: #00ff00; --tui-warning: #ffff00; --tui-danger: #ff0000; --tui-info: #00ffff; } /* Use it */ <html data-theme="cyberpunk">

JavaScript Theme Switching

Use the theme utility for programmatic control:

import { initTheme } from 'web-tui/js'; // Initialize with options const theme = initTheme({ defaultTheme: 'dark', storageKey: 'my-app-theme', // localStorage key onChange: (newTheme) => { console.log('Theme changed to:', newTheme); } }); // Get current theme console.log(theme.getTheme()); // 'dark' // Set theme theme.setTheme('matrix-green'); // Toggle between themes theme.toggle(['dark', 'light']); // Get available themes console.log(theme.getAvailableThemes()); // ['dark', 'light', 'matrix-green', 'retro-amber', ...]

Or use vanilla JavaScript:

// Simple theme toggle function setTheme(name) { document.documentElement.setAttribute('data-theme', name); localStorage.setItem('theme', name); } // Load saved theme on page load const savedTheme = localStorage.getItem('theme') || 'dark'; setTheme(savedTheme);

System Preference Detection

Automatically detect user's system color scheme preference:

// Detect system preference const prefersDark = window.matchMedia('(prefers-color-scheme: dark)'); function setThemeFromSystem() { const theme = prefersDark.matches ? 'dark' : 'light'; document.documentElement.setAttribute('data-theme', theme); } // Set initial theme setThemeFromSystem(); // Listen for changes prefersDark.addEventListener('change', setThemeFromSystem);
