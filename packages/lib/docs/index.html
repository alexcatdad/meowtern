>_ web-tui

A framework-agnostic terminal UI component library. Build retro terminal interfaces with pure CSS classes, similar to DaisyUI.

Get Started Components GitHub

Why web-tui?

Framework Agnostic

Works with any framework or vanilla HTML. Use with React, Vue, Svelte, or no framework at all.

CSS-Only Interactive

Tabs, modals, dropdowns, and more - all without JavaScript. Uses native CSS states like :checked and :focus-within.

7 Terminal Themes

Dark, Light, Matrix Green, Retro Amber, Polar Night, and more. Easy custom theme creation with CSS variables.

Tailwind Compatible

Use as a Tailwind plugin for terminal color utilities, or use standalone CSS for non-Tailwind projects.

50+ Components

Windows, boxes, buttons, tables, tabs, menus, dialogs, progress bars, badges, and more terminal UI elements.

React Components

Optional React components available for those who prefer a component-based approach.

Quick Start

  1. Install the package
    npm install web-tui
  2. Import the CSS
    <link rel="stylesheet" href="node_modules/web-tui/src/css/web-tui.css">

    Or in your CSS/JS:

    @import 'web-tui/css';
  3. Use the classes
    <div class="tui-window"> <div class="tui-window-header"> <span class="tui-window-title">Terminal</span> </div> <div class="tui-window-body"> <p class="tui-text">Hello, World!</p> </div> </div>

Usage Options

<!-- Just include the CSS file --> <link rel="stylesheet" href="web-tui/css"> <!-- Then use classes --> <button class="tui-btn tui-btn-primary">Click Me</button>
// tailwind.config.js module.exports = { plugins: [ require('web-tui/plugin') ] } // Then use terminal colors <div class="bg-terminal-panel text-terminal-cyan">...</div>
import { TerminalWindow, Box, Button } from 'web-tui/react'; function App() { return ( <TerminalWindow title="My App"> <Box bordered title="Welcome"> <Button variant="primary">Click Me</Button> </Box> </TerminalWindow> ); }
import { initTheme, initDialog } from 'web-tui/js'; // Initialize theme switcher const theme = initTheme({ defaultTheme: 'dark' }); theme.setTheme('matrix-green'); // Initialize dialog (for enhanced behavior) const dialog = initDialog('.my-dialog'); dialog.open();
