What is web-tui?
web-tui is a framework-agnostic terminal UI library. It works with any framework or vanilla HTML/CSS, similar to DaisyUI.
How does it work?
It uses pure CSS for styling and state management. Interactive components use the :checked pseudo-class, focus-within, and native HTML elements like <details>.
Is JavaScript required?
No! All components work without JavaScript. Optional JS utilities are available for enhanced functionality like keyboard navigation and command palettes.
